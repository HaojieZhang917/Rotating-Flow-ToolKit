# Lopez 模型双模态网格无关性验证

## 1. 验证工况

本验证针对 Lopez 广义 Boussinesq 模型在以下工况下的两个空间中性模态：

```math
T_w=1.08,\qquad \omega=0,\qquad N=49,59,69,79,89.
```

所有网格均采用 `CRD_BF.Cheb` 的 rational Chebyshev 坐标。对每个 `N` 均重新：

1. 在参考临界点附近求解固定 `beta` 的中性 Reynolds 数；
2. 用两轮三点抛物线拟合校正 `(R_c,beta_c)`；
3. 在校正后的临界点重新求解空间特征值；
4. 将完整特征函数插值到共同的 `z in [0,20]` 网格；
5. 经过幅值归一化和复相位对齐后，与 `N=89` 比较形状误差。

中性点校正容限为 `1e-8`，多项式特征值问题自身的残差约为
`1e-11` 到 `1e-12`。

特征函数误差随配置点数增加持续减小，两个模态均表现出清晰的网格收敛趋势。

## 5. 关于误差非严格单调

临界参数在 `N>=59` 后已经进入 `1e-5` 到 `1e-7` 的小量变化区间。此时局部
抛物线拟合误差、中性根容限以及 IAR 特征值容限与空间离散误差处于相近量级，因而
`R_c` 或 `beta_c` 的最后几位不必随 `N` 严格单调变化。这是数值误差平台，不代表
网格发散。

判断网格无关性应综合使用：

- 临界参数相对变化；
- 中性残差和多项式特征值残差；
- 经过相位对齐的特征函数形状误差。

## 6. 结论

生产计算使用的 `N_cheb=69` 对两个模态均达到网格无关：

- Type-I 临界参数相对误差不超过 `5.5e-6`，速度模态误差约 `0.021%`；
- Type-II 临界参数相对误差不超过 `3.4e-7`，速度模态误差约 `0.0066%`；
- 所有中性残差均小于 `7e-9`；
- 所有多项式特征值残差均小于 `7e-11`。

因此继续将配置点数从 `69` 增加到 `79` 或 `89` 不会对当前中性曲线结论产生可见
改变，`N_cheb=69` 可以作为后续 Lopez 模型中性曲线计算的生产网格。

原始数值数据见 `lopez_grid_independence_Tw1.08.tsv`，可重复计算脚本为
`LopezGridIndependence.jl`。
